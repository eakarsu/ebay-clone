## Getting Started

### Creating an Account

1. Click **"Register"** in the top navigation
2. Enter your email, username, and password
3. Verify your email address
4. Complete your profile

### Logging In

1. Click **"Sign In"**
2. Enter your email and password
3. (Optional) Use Two-Factor Authentication for extra security

### Setting Up Two-Factor Authentication (2FA)

1. Go to **Profile → Security Settings**
2. Click "Enable 2FA"
3. Scan the QR code with an authenticator app (Google Authenticator, Authy)
4. Enter the 6-digit code to verify
5. Save your backup codes in a safe place

## Bidding & Auctions

### How Auctions Work

1. Seller sets starting bid and auction duration
2. Buyers place bids
3. Highest bidder at end wins
4. Winner pays and seller ships

### Placing a Bid

1. Enter your maximum bid amount
2. Click **"Place Bid"**
3. System auto-bids up to your max

### Proxy Bidding (Automatic Bidding)

- Enter the **maximum** you're willing to pay
- System bids the minimum needed to stay ahead
- You only pay slightly more than the second-highest bidder

**Example:**
- Item starting bid: $10
- You bid max: $50
- Someone bids: $15
- System bids for you: $16 (one increment)
- You're still winning at $16 (not $50)

### Bid Increments

| Current Price | Bid Increment |
|---------------|---------------|
| $0.01 - $0.99 | $0.05 |
| $1.00 - $4.99 | $0.25 |
| $5.00 - $24.99 | $0.50 |
| $25.00 - $99.99 | $1.00 |
| $100.00 - $249.99 | $2.50 |
| $250.00 - $499.99 | $5.00 |
| $500.00 - $999.99 | $10.00 |
| $1000+ | $25.00 |

### Reserve Price

- Some auctions have a hidden minimum price
- "Reserve not met" means bidding hasn't reached it
- Item won't sell unless reserve is met

### Buy It Now + Auction

- Some auctions offer instant purchase option
- Buy It Now disappears after first bid
- Can choose to bid or buy immediately

### Bid Retraction

**Allowed reasons:**
- Typed wrong bid amount
- Item description changed significantly
- Cannot contact seller

**Navigate to:** My eBay → Bid Retractions

### Proxy Bidding (Automatic Bidding)

**For Auctions**

#### How to Set Up Proxy Bidding

1. Go to auction listing
2. Click **"Place Bid"**
3. Enter your **maximum bid** (highest you'll pay)
4. System automatically bids for you

#### How It Works

- System bids minimum increment needed to win
- Increases automatically when outbid (up to your max)
- You only pay one increment above second-highest bidder

**Example:**
```
Your max bid: $100
Current bid: $25
System bids: $26 for you
Someone bids: $50
System bids: $51 for you
You win at: $51 (not $100)
```

#### Managing Proxy Bids

1. Go to **My eBay → Bids**
2. View all active proxy bids
3. Increase maximum (cannot decrease)
4. Cancel bid (with valid reason)
